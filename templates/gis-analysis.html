Array mit den Filtern die angewendet werden sollen. Aufbau:
id: ID des Attributs das verglichen werden soll. Falls das Attribut (aktuell nur geometry) ein PostGIS-Feld ist, sollte zusätzlich func übergeben werden.
comp: Funktion für die Abfragen, z.B. > oder <
values: Array von Werten für die Abfrage mit comp. Meistens nur ein Wert. Wenn func gesetzt ist, können auch mehrere Werte vorkommen.
func: Falls eine (PostGIS)-Funktion auf die Spalte angewendet werden soll, bevor sie mit comp verglichen wird. Unterstützt werden aktuell distance und area.

Array mit IDs von Kontexttypen, für die die Filter angewendet werden sollen. Keins/ein leeres Array sucht in allen Typen.

Array mit Spalten auf die das Ergebnis eingeschränkt werden soll. Keins/ein leeres Array gibt alle Spalten zurück.
